# 门诊挂号操作指南（样例 · 虚构）

> 本文档为 BizMind Demo 虚构 HIS 风格手册，非真实医院制度。

## 1. 适用范围

适用于本院门诊窗口及自助机挂号场景。

## 2. 患者需携带证件

- 中国大陆居民：**居民身份证**或**医保卡**
- 外籍患者：**护照**
- 儿童：监护人身份证 + 户口本或出生医学证明

## 3. 挂号流程

1. 患者到分诊台或自助机选择科室
2. 核验身份信息
3. 选择号源（普通 / 专家）
4. 支付挂号费（支持微信、支付宝、医保卡）
5. 打印挂号单，到候诊区等候叫号

## 4. 退号规则

- 就诊日前一天 18:00 前可免费退号
- 当日退号收取 50% 挂号费

## 5. 权限说明

| 角色 | 权限 |
|------|------|
| 挂号员 | 窗口挂号、退号 |
| 分诊护士 | 分诊、加号（需主任授权） |
| 系统管理员 | 号源维护、权限配置 |

## 6. 常见问题

**Q：忘带身份证能否挂号？**  
A：可输入身份证号人工核验，但建议随身携带证件以免延误。

**Q：专家号能否转诊？**  
A：需原科室主任同意，由挂号室办理转号。
