# 跨境电商库存盘点 SOP（样例 · 虚构）

> BizMind Demo 虚构 ERP 文档。

## 1. 盘点周期

- **全面盘点**：每月最后一个工作日
- **循环盘点**：每周对 A 类 SKU 抽检 10%
- **大促前**：活动开始前 3 天加盘一次

## 2. 盘点前准备

1. 冻结 WMS 中待盘点仓位的出入库（盘点单创建后自动冻结）
2. 导出系统账面库存
3. 打印盘点表或使用 PDA 扫码清单

## 3. 盘点步骤

1. 两人一组：一人清点、一人复核
2. 按库位顺序扫码，记录实盘数量
3. 差异超过 ±2% 或金额超过 500 元须复盘
4. 盘点完成后 24 小时内提交盘点报告

## 4. 差异处理

| 差异类型 | 处理方式 |
|----------|----------|
| 盘盈 | 查明原因，经仓储主管审批后调账 |
| 盘亏 | 填写《库存差异单》，3 个工作日内结案 |
| 批次错误 | 联系报关组核对 HS 编码与批次 |

## 5. 报关相关

跨境 SKU 盘点须同时核对：

- 海关备案料号
- 保税账册余量
- 进销存平衡表

## 6. 责任人

- 仓储主管：组织盘点、审批差异
- 财务：监督调账
- 系统管理员：WMS 冻结与解锁
